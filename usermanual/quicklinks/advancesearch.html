What is advance search?

Advance search is a feature which allows quick access to activities/actions in MifosX
e.g. create client, create loan product etc.

To access advance search, mouse hover on left side section of the application it will bring up list of quick links, click on advance search from the list.


How does it work?

Once the user start entering the key words in advance search results will be displayed based on the entered word.
e.g. If key word "create" is entered then it will list all available create activities/actions like create client, create group, create loan product.
