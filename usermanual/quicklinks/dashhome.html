What is Dash Home?

Dash Home (a.k.a Expert Search) allows you to search activities within application for easier access.
e.g. create client, create loan product etc.



To start using the Dash Home, mouse hover on left side section in the application, select Dash home from quick links.


How Dash Home works?

Based on entered keywords, quickly searches all matching activities that can be done within the application.
e.g. key word "create" suggests you

  • create client
  • create group
  • create loan product
  • etc...


What are Recent Activities?

Shows eight (8) most recently done actions. If user wants to repeat any of these actions just need to click on icon displayed


What are Frequent Activities?

Displays eight (8) most frequently done actions by the user for quick access.

This will act as user specific “Quick navigation menu”.
